# Codey

Codey is a Python package providing AI tooling.

## Installation

```bash
pip install .
```

## Usage

After installation, run the CLI with:

```bash
codey
```

## Configuration

Set environment variables or use a local `.env` file:

- `OPENAI_API_KEY` (required)
- `MODEL_NAME` (optional)
- `PROMPT_PATH` (optional)
